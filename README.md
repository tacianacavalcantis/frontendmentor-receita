# What I Learned / O Que Aprendi

**EN:**
During this project, I reinforced my knowledge of:

1. Semantic HTML - Using the correct tags ('<main>, <section>, <header>, <footer>') for better accessibility and SEO.

2. CSS Organization - Structuring stylesheets with comments and using utility classes (like .margin-15) to avoid code duplication.

3. Responsive Design - Implementing media queries to ensure the page looks good on all devices.

4. Table Accessibility - Using <th> with scope attributes and proper table structure for screen readers.

5. CSS Variables - Working with HSL color values for consistent theming.

**PT:**

Durante este projeto, eu refinei meus conhecimentos em:

1. HTML Semântico - Uso das tags corretas <main>, <section>, <header>, <footer> para melhor acessibilidade e SEO.

2. Organização CSS - Estruturação de folhas de estilo com comentários e uso de classes utilitárias (como .margin-15) para evitar duplicação de código.

3. Design Responsivo - Implementação de media queries para garantir que a página fique boa em todos os dispositivos.

4. Acessibilidade em Tabelas - Uso de <th> com atributos scope e estrutura adequada de tabela para leitores de tela.

5. Variáveis CSS - Trabalho com valores de cor HSL para temas consistentes.
